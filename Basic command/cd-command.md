# apa itu command ( cd )
cd ( change directory ) sesuai artinya kita berarti bisa berpindah/masuk directory,  
pemakaian ini umum banget di antarmuka CLI ( command line interfaces ) untuk kita berpindah pindah directory/folder  

## Pemakaian command cd
## cd _nama_directory_ = untuk berpindah pada directory/folder yang ada di directory kerja kita
contohnya disini saya ingin masuk ke folder atau directory folder_belajar ( yang ada di directory saya berada )
