# apa itu command ( cat )
cat ( concatenate ) yang artinya menggabungkan, nah command cat ini biasanya paling umum digunakan untuk membaca dan menampilkan isi file di terminal
adapun fungsi lainnya yaitu bisa digunakan untuk 
- Menggabungkan file ( sesuai artinya )
- Membuat file baru

## Pemakaian command cat 
## cat _nama_file_ = untuk membaca sekaligus menampilkan isi file 
contoh nama file nya disini belajar.txt
<img width="1488" height="454" alt="cat belajar txt" src="https://github.com/user-attachments/assets/d5677518-5e7e-4ded-8442-1adb8e3db435" />

## cat _nama_file_ _nama_file_ = untuk membaca dan menampilkan isi file beberapa file sekaligus
contoh disini kita akan membaca dan menampilkan 2 file sekaligus yaitu belajar.txt belajar2.txt 
<img width="1488" height="492" alt="cat 2 file" src="https://github.com/user-attachments/assets/09614058-d20a-4861-be47-2d2afb731242" />

## cat -n _nama_file_ = untuk membaca dan menampilkan isi file dan diberikan nomor pada setiap baris isi filenya ( supaya lebih jelas )
contoh disini kita akan membaca dan menampilkan file belajar4.txt dengan memberikan option ( -n ) maka setiap baris diberi nomor 
<img width="1488" height="602" alt="cat -n" src="https://github.com/user-attachments/assets/1a512728-f251-4d00-85b7-13a7c90e1578" />

## cat _nama_file__nama_file_ > _file_baru_ = perintah untuk membuat file baru dengan menggabungkan file file yang sudah ada
contoh disini kita buat file baru namanya belajarcombo.txt, yaitu hasil dari penggabungan file belajar2.txt dan belajar3.txt, kemudian disini langsung kita baca file barunya ( belajarcombo.txt ) dan hasilnya menampilkan isi file dari file file yang kita gabungkan
