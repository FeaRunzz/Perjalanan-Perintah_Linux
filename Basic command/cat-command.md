# apa itu command ( cat )
cat ( concatenate ) yang artinya menggabungkan, nah command cat ini biasanya paling umum digunakan untuk membaca dan menampilkan isi file di terminal
adapun fungsi lainnya yaitu bisa digunakan untuk 
- Menggabungkan file ( sesuai artinya )
- Membuat file baru

## Pemakaian command cat 
## cat _nama_file_ = untuk membaca sekaligus menampilkan isi file 
contoh nama file nya disini belajar.txt
